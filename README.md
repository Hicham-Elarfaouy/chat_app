#  Social App ✨


Social is open-source  app for Android & ios with Firebase. It is built with Flutter Framework.

Star ⭐ the repo if you like it.

<p><img src="snapshots/HomeScreen.png" height="500" /></p>



## Features

 - Login
 - Register
 - Create Posts
 - like & comment in Posts
 - Chat with friends
 - Search for friend
 - Edit profile & cover
 - Settings
